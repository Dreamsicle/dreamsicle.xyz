Pointer

· Dreamsicle Developer Hub

Summary
Pointer is a Discord bot designed to be used as a base for other bots. It has an updating system built in, and has few commands on purpose. It is under the BSD 3-Clause license.

Pre-built functions

At the time of writing, Pointer has one function available that's exclusive to it - sendMessage(embedTitle, embedDescription, messageType). It sends the data presented to it as an embed in Discord. It is a shorthand for Discord.JS' send function.

embedTitle

embedTitle is a string that is sent as the title of the Discord embed.

embedDescription

embedDescription is a string that is sent as the body text of the Discord embed.

messageType

Optional

At the time of writing, messageType simply changes the color of the message if it's "normal" (command went a-okay, assumed if messageType is not defined) or an "error" (sets embed color to Dreamsicle red).

There are no plans, but it will be used for other things in the future.
